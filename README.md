## About The Project

This is a linear regression model I made in Python using Pandas, matplotlib, and NumPy to investigate  the [Environmental Dataset - UN Data](https://www.kaggle.com/datasets/vineethakkinapalli/united-nations-environment-data?resource=download). This is currently a work in progress. 

<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

You will need to install:
- Pandas
- matplotlib
- Numpy

Any version is okay. 

I used VS Code as the IDE for this code. This can be installed [here](https://code.visualstudio.com/download?_exp_download=fb315fc982).


<!-- USAGE EXAMPLES -->
## Usage

This project can be used to view the dataset 'Carbon Dioxide Emission Estimates' (part of the [Environmental Dataset - UN Data](https://www.kaggle.com/datasets/vineethakkinapalli/united-nations-environment-data?resource=download)) graphically. 

By changing the commented out sections, you can view this as a scatter plot, a line graph, or a forecast. The default set up is a forecast.
